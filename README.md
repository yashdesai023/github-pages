## Steps to Reproduce
1. Created `index.html` file with the webpage structure.
2. Created `style.css` for a modern dark theme styling.
3. Created this `README.md` file.
4. Pushed it to a new GitHub repo.
5. Went to repo setting → Pages → enabled GitHub Pages.
6. Selected the `main` branch and root folder.
7. Site was published and live link verified.
